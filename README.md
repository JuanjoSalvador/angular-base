# angular-base
Angular 1.6.3 base project.

# THIS PROJECTS IS DEPRECATED.
This is legacy code. Consider using Angular 5 instead of AngularJS
---

### Includes

* Angular 1.6.3
* Font Awesome
* Angular UI Router
* Bootstrap 3.3.7


### How to install

    git clone https://github.com/JuanjoSalvador/angular-base.git
    npm install
    
### How to run it

    npm start
