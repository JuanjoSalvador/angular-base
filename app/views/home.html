Angular Base 0.0.3

Desarrollado por Juanjo Salvador

¡Bienvenido a Angular Base 0.0.3!

Angular Base 0.0.3 es una plantilla de proyecto de AngularJS pensada para agilizar el desarrollo de aplicaciones web.

Incluye los siguientes componentes

  • Angular 1.6
  • Font Awesome
  • AngularUI Router
  • Bootstrap 3
  • SASS (opcional)

¿Cómo se usa?

Angular Base mantiene tus scripts organizados en carpetas según sean directivas, servicios, o controladores, manteniendo una estructura sencilla de carpetas y scripts.

En el index.html encontramos incluidas todas las dependencias que necesitas. Igual que en app.js ya encontramos inyectadas las dependencias para no tener que hacerlo manualmente, así como una serie de rutas definidas. En resumen: tenemos todo listo para solamente arrancar y empezar a desarrollar.
